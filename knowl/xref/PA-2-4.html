Preview Activity 2.4.1.

Considera la función \(f(x) = \tan(x)\text{,}\) y recuerda que \(\tan(x) = \frac{\sin(x)}{\cos(x)}\text{.}\)
  1. ¿Cuál es el dominio de \(f\text{?}\)
  2. Usa la regla del cociente para mostrar que una expresión para \(f'(x)\) es
    \begin{equation*} f'(x) = \frac{\cos(x)\cos(x) + \sin(x)\sin(x)}{\cos^2(x)}\text{.} \end{equation*}
  3. ¿Cuál es la Identidad Trigonométrica Fundamental? ¿Cómo se puede usar esta identidad para encontrar una forma más simple de \(f'(x)\text{?}\)
  4. Recuerda que \(\sec(x) = \frac{1}{\cos(x)}\text{.}\) ¿Cómo podemos expresar \(f'(x)\) en términos de la función secante?
  5. ¿Para qué valores de \(x\) está definida \(f'(x)\text{?}\) ¿Cómo se compara este conjunto con el dominio de \(f\text{?}\)
