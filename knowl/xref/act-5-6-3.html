Activity 5.6.4.

Considera las funciones \(f(x) = 2-x^2\text{,}\) \(g(x) = 2-x^3\text{,}\) y \(h(x) = 2-x^4\text{,}\) todas en el intervalo \([0,1]\text{.}\) Para cada una de las preguntas que requieren una respuesta numérica a continuación, escribe tu respuesta exactamente en forma de fracción.
  1. En los tres conjuntos de ejes proporcionados en Figura 5.6.9, dibuja un gráfico de cada función en el intervalo \([0,1]\text{,}\) y calcula \(L_1\) y \(R_1\) para cada una. ¿Qué observas?
  2. Calcula \(M_1\) para cada función para aproximar \(\int_0^1 f(x) \,dx\text{,}\) \(\int_0^1 g(x) \,dx\text{,}\) y \(\int_0^1 h(x) \,dx\text{,}\) respectivamente.
  3. Calcula \(T_1\) para cada una de las tres funciones, y por lo tanto calcula \(S_2\) para cada una de las tres funciones.
  4. Evalúa cada una de las integrales \(\int_0^1 f(x) \,dx\text{,}\) \(\int_0^1 g(x) \,dx\text{,}\) y \(\int_0^1 h(x) \,dx\) exactamente usando el Primer FTC.
  5. Para cada una de las tres funciones \(f\text{,}\) \(g\text{,}\) y \(h\text{,}\) compara los resultados de \(L_1\text{,}\) \(R_1\text{,}\) \(M_1\text{,}\) \(T_1\text{,}\) y \(S_2\) con el valor verdadero de la integral definida correspondiente. ¿Qué patrones observas?
